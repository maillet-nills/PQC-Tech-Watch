## Description
**Symmetric Cryptography** (or *Symmetric Key Cryptography*), is an encryption system where the sender and the receiver of a message use a single common key to encrypt messages.

- **Examples:** AES, ChaCha20 
- **Based on:** shared secret key
- **Quantum threat:** ⚠️ Grover's algorithm halves security → mitigated by doubling key size
