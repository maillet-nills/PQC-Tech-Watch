## Description
**Hash functions** do not require a key, they are using mathematical algorithms to convert messages of any arbitrary length into a fixed-length output (hash value/ digest).

- **Examples:** SHA-256, SHA-3 
- **Based on:** one-way functions 
- **Quantum threat:** ⚠️ Grover's algorithm halves security → mitigated by doubling output size
